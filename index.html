Coquette Coding

Αρχική σελίδα

Καλώς ορίσατε στην ιστοσελίδα Coquette Coding,στην οποία θα βρείτε ότι χρεάζεστε για να μάθετε να προγραμματίζετε σε Java.

Στόχος της σελίδας μας είναι να μπορείτε να βρείτε ότι πληροφορία και βοήθεια χρειάζεσται καθώς μαθένετε να προγραμματίζετε σε Java.

Ο ιστοχώρος μας αποτελείται από:

  • Ανακοινώσεις: Στη σελίδα αυτή, θα βρείτε τις τελευταίες ανακοινώσεις και ενημερώσεις άλλα σημαντικά νέα. Να έχετε τον νου σας!
  • Επικοινωνία: Εδώ μπορείτε να στείλετε τις απορείς και ό,τι σας απασχολεί σχετικά με τα μαθήματά μας.
  • Έγγραφα Μαθήματος: Στην σελίδα αυτή θα βρείτε απαραίτητα έγγραφα όπως εκπαιδευτικό υλικό και βοηθητικούς οδηγους για την εκμάθηση της γλώσσας Java.
  • Εργασίες: Εδώ θα ανεβαίνουν οι εργασίες και τα προβλήματα προς επίλυση που θα παραδίδονται στο μάθημα ώστε να μπορείτε να τα δείτε και από το σπίτι σας.
